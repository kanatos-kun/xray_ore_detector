# Feature
Add an enchantment to the pickaxe that add xray to nearby ore near the player.
The ore has different color to differentiate the different kind of ore
![glowing ore when having the xray detector enchantment](https://cdn.modrinth.com/data/cached_images/ca092c696024ff47fe77b667b67b1af46bd44e48.png)
can xray :
- copper ore
- iron ore
- emerald ore
- gold ore
- diamond ore
- redstone
- coal 
- ancient debris
# Installation
- Download the file
- open minecraft launcher
- create a new world
- in the **more** tab click the **datapack button**
- copy paste the downloaded file in the directory
